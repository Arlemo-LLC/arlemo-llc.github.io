Why this exists

Most string tension tools are fine for standard tunings and clumsy for weird ones. Strings is built around the practical question players actually have: what should I put on this instrument today?

How it works

The calculator converts each note to frequency, uses scale length and string unit weight to estimate pounds of tension, then chooses the closest available gauge for the selected feel target.

Buying links

The buying flow will favor close stock sets first and custom singles when the tuning needs them. We verify product IDs before turning on direct cart links.
